Preguntas Frecuentes

Resolvemos tus dudas antes de empezar.

¿Cuánto tardáis en entregar la web terminada?

+

Habitualmente entregamos la web completamente funcional en un plazo de 10 a 14 días laborables desde que recibimos todo el material (tus textos, imágenes y logotipo). Mantenemos una comunicación fluida para que veas los avances.

¿Tengo que pagar el mantenimiento obligatoriamente?

+

No, el mantenimiento es 100% opcional. Puedes alojar y gestionar la web por tu cuenta tras la entrega. Sin embargo, nuestro plan de mantenimiento está pensado para quitarte dolores de cabeza: incluye un servidor premium ultrarrápido, copias de seguridad diarias y cambios de contenido ilimitados para que tú solo te ocupes de tu negocio.

Ya tengo una web, ¿podéis aprovecharla y mejorarla?

+

¡Por supuesto! Es nuestro servicio más demandado (Rediseño Pro). Hacemos una auditoría completa de tu web actual, salvamos el contenido que funciona y reescribimos el código y el diseño para multiplicar la velocidad de carga y mejorar el porcentaje de conversión de visitantes a clientes.

¿Cómo funciona la forma de pago?

+

Trabajamos con un sistema transparente y seguro: pedimos un 50% del presupuesto al iniciar el proyecto para reservar los recursos y empezar a diseñar, y el 50% restante solo cuando la web está terminada, revisada por ti y lista para publicarse.

¿Mi web estará optimizada para salir en Google?

+

Sí. Todas nuestras webs se entregan con SEO On-Page básico integrado. Esto significa que la estructura del código, las etiquetas, los metadatos y la velocidad de carga están diseñados exactamente como a Google le gusta para que empieces a posicionar localmente desde el primer día.
